# 电脑内存清理（整蛊版）- 设计文档

## 概述

一个 Windows 桌面小工具，点击后假装清理电脑，实际是3秒后强制关机。用于整蛊朋友/同事。

## 技术栈

- **语言：** Python 3.x
- **GUI框架：** PyQt5
- **打包工具：** PyInstaller
- **输出：** 单文件 EXE

## 界面设计

### 窗口属性

- 标题：「电脑内存清理」
- 窗口尺寸：400 x 300 像素（固定，不可拉伸）
- 居中显示，无最大化按钮
- 关闭按钮直接退出程序

### 阶段一：主界面

- 顶部标题：「电脑内存清理」（大字体，居中）
- 副标题：「一键清理系统垃圾，释放内存空间」
- 中央大按钮：「立即清理」（科技感蓝底白字，圆角）
- 按钮尺寸：160 x 50 像素

### 阶段二：清理进度（点击按钮后切换）

- 标题变为：「正在清理中...」
- 进度条（PyQt5 QProgressBar）
  - 风格：块状进度条，深蓝填充
  - 最小值0，最大值100
- 动态文字说明（QLabel）：
  - 0-30%：「AI大模型正在分析系统垃圾文件」
  - 30-70%：「AI大模型正在分析多余应用」
  - 70-100%：「生成清理方案」
- 进度条下方显示百分比数字

### 阶段三：关机倒计时弹窗（QMessageBox）

- 进度完成后弹出
- 标题：「警告」
- 倒计时文字（大字体居中）：
  - 第1秒：「三秒后关机」
  - 第2秒：「二秒后关机」
  - 第3秒：「一秒后关机」→ 立即执行关机

## 关机实现

调用 Windows 命令：
```
shutdown /s /t 0
```
（`/s` 关机，`/t 0` 立即执行）

## 打包配置

- 命令：`pyinstaller --onefile --noconsole cleanup.exe`
- `--noconsole`：不显示CMD窗口
- 输出单文件 EXE

## 文件结构

## 依赖

- PyQt5

## 验收标准

1. 双击EXE打开窗口，显示"电脑内存清理"标题和"立即清理"按钮
2. 点击按钮后，进度条动态跑完，文字按阶段切换
3. 进度完成后弹出倒计时弹窗，每秒切换文字
4. 倒计时结束后电脑自动关机
