# 广告跳过恶搞工具 - 设计规格

## 1. 概念与愿景

一款看起来非常正经的广告跳过工具，但实际上就是个恶搞程序。用户选择平台、填写广告时长后点击"跳过"，会显示一连串专业的假进度（获取接口→破解算法→绕过广告），同时倒计时器同步进行，最后弹出"广告跳过成功"——而实际上什么都没发生。

**核心体验：** 界面越正经，恶搞效果越好。

## 3. 用户流程

```
启动 → 显示主界面
       ↓
   选择平台（必选）+ 填写时长（默认15秒）
       ↓
   点击「开始跳过广告」
       ↓
   ┌─ 显示进度区域，隐藏输入区域
   │  ├─ 进度条从0% → 100%（与倒计时同步）
   │  ├─ 状态文字分阶段切换
   │  │   - 0-30%：「正在获取平台广告接口...」
   │  │   - 30-60%：「正在破解广告加密算法...」
   │  │   - 60-90%：「正在绕过广告验证...」
   │  │   - 90-100%：「广告跳过成功！」
   │  └─ 倒计时显示剩余秒数（小数点后2位）
   │
   └─ 倒计时结束
       ↓
   显示「✅ 广告已成功跳过！」成功气泡
       ↓
   2秒后自动恢复到初始状态（可再次使用）
```

## 4. 状态管理

### UI 状态

| 状态 | 输入区域 | 进度区域 | 主按钮 |
|------|----------|----------|--------|
| 初始 | 可见 | 隐藏 | "开始跳过广告" |
| 执行中 | 隐藏 | 可见 | 禁用 |
| 完成 | 隐藏 | 可见（100%） | "开始跳过广告" |
| 恢复 | 可见 | 隐藏 | "开始跳过广告" |

### 倒计时精度

- QTimer 间隔：100ms（精度 0.1 秒）
- 显示：小数点后 2 位
- 总时长：用户输入的秒数

## 6. 技术实现

- **语言：** Python 3
- **GUI框架：** PyQt5
- **文件：** `ad_skip.py`（单文件）
- **窗口尺寸：** 500 × 550 px
- **窗口标题：** "广告跳过助手"

### 关键组件

| 组件 | Qt 类 |

| 主窗口 | QWidget |
| 应用 | QApplication |
| 布局 | QVBoxLayout, QHBoxLayout, QGridLayout |
